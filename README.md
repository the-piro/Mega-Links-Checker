## VPS Deployment 

1. Clone the repository:
    ```sh
    git clone https://github.com/XalFH/Mega-Links-Checker.git
    cd Mega-Links-Checker
    ```

2. Install dependencies:
    ```sh
    pip install -r requirements.txt
    ```

3. Run bot

```sh
python3 main.py

```
